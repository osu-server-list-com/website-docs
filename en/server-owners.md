##### ðŸ“‹ Filling Out the Form

**Server Name**
- Enter your server's name
- **No spaces allowed** â€” use hyphens (`-`) or underscores (`_`) instead
- Example: `My-Cool-Server` or `MyCoolServer`
- This name will be displayed across the platform

**Server URL (without protocol)**
- Enter your production server URL **without** `http://` or `https://`
- Example: `play.myserver.com` or `myserver.net`
- This is the main URL players will use to access your server

**Devserver (without protocol)**
- Enter your development/staging server URL **without** `http://` or `https://`
- Example: `dev.myserver.com` or `staging.myserver.net`
- If you don't have a separate dev server, you can enter the same URL as your main server

**Categories**
- Select all categories that apply to your server
- Available categories include:
  - **AntiCheat** â€” Strong anti-cheat enforcement
  - **AutoPilot** â€” Automated features
  - **Catch** â€” osu!catch focused
  - **Mania** â€” osu!mania focused

  - **Standard** â€” osu!standard focused
  - **Taiko** â€” osu!taiko focused
  - And more...
- You can select multiple categories that best describe your server

**Logo Image**
- Upload a **PNG file only**
- Square images (1:1 aspect ratio) work best
- Recommended size: **256x256px** or larger for clarity
- Will be displayed as **64x64px** with rounded corners
- Make sure your logo is clear and recognizable at small sizes
